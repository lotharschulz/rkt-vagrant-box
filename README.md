Vbox to test coreos/rocket
--------------------------

preconditions:

- unix-like operating system
- virtual box
- vagrant
- ansible

usage:

- vagrant up
- vagrant ssh
-  cd rocket-v0.3.1
-  ./rkt help
-  ./rkt list

further coreos rocket reading:

- [rocket](https://github.com/coreos/rocket/blob/master/README.md)
